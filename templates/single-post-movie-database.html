The Movie Database was built to provide users with an intuitive and interactive platform to discover and explore films. Its purpose was to combine robust functionality with an easy-to-use interface, making it simple for users to search, filter, and view detailed movie information.

Lesson learned: The project aimed to demonstrate proficiency in React, JavaScript, and API integration while creating a responsive and dynamic application. Key outcomes include an optimized search and filter system, modular component design, and smooth user interactions that enhance both usability and performance.

Design process

Wireframes

The Movie Database project was designed to provide users with a simple, intuitive interface for discovering and exploring films. Using wireframes, I mapped out the primary user flows, including searching, filter, and viewing detailed movie information. The goal was to ensure that the interface remained clean and accessible while maintaining a visually engaging layout that emphasized the content hierarchy and user interaction paths. This planning stage helped identify key UI components and prepared the foundation for the front-end implementation in React.

Problem/Solution

The main challenge was integrating multiple APIs and presenting large amounts of movie data without overwhelming the user. To address this, I implemented a modular component structure that separates the search, filter, and detail views, making the interface easier to navigate. Lazy loading techniques and pagination were added to optimize performance and reduce initial load times. While I didn’t create full error-state screens, I did design a clear empty-state message for the Favourites tab, ensuring users understood when no movies had been added yet. This helped maintain a smooth and intuitive browsing experience.

Functionality focus

Functionality was a key focus of this project, particularly the implementation of dynamic API calls, responsive search functionality, and interactive filtering. React hooks were used for state management, allowing smooth updates as users refined their queries. Key features include a responsive grid layout, interactive movie cards with hover effects, and modal dialogs for detailed movie information. These elements combined to create a robust, interactive, and engaging application that demonstrates both coding proficiency and thoughtful UX design.
