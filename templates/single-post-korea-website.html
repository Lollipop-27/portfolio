The Korea Website was designed to showcase Korean culture, travel, and design aesthetics through a visually engaging and accessible interface. The project emphasizes clean layouts, harmonious typography, and thoughtful use of color to convey both information and cultural context.

Project Goals: The goals were to create a user-friendly website with a strong visual identity, highlighting layout, typography, and UX choices. The project focused on responsive design, accessibility, and creating an engaging gallery system to present images, colors, and design elements effectively.

Live Site GitHub

Design process

Colour palette

The website’s color palette was carefully chosen to balance modern web aesthetics with subtle nods to traditional Korean culture. Soft neutrals provide a clean and accessible background, while bold accent colors—such as deep reds and vibrant blues—highlight key elements and create visual hierarchy. Typography and spacing were designed to complement the palette, ensuring readability and maintaining a harmonious, cohesive look across all pages.

Gallery Design Variations

To showcase the visual content effectively, multiple gallery layouts were developed, including grid-based sections, full-width images, and carousel slideshows. Each variation emphasizes responsiveness and accessibility while providing engaging ways for users to explore scenic photography, cultural highlights, and design details. These gallery options allow flexibility in presenting content while maintaining a consistent visual language throughout the site.
