The Capstone Vacation Rental project was a collaborative effort aimed at building a fully functional, user-friendly booking website. The team’s goal was to combine strong front-end design with back-end functionality, creating a platform that allows users to browse listings, filter by amenities, and complete bookings seamlessly. Each team member contributed their expertise to ensure a cohesive product that met both user needs and technical requirements.

Lesson learned: Through this project, we learned the importance of clear communication, task delegation, and iterative problem-solving. Coordinating as a team highlighted the value of version control, code reviews, and collaborative planning. Additionally, balancing design consistency with technical feasibility reinforced practical skills in project management and teamwork.

Design process

Communication

Effective communication was critical to the success of this project. We established regular meetings, shared progress updates via a project management tool, and used collaborative platforms to discuss design and code decisions. This ensured alignment on both milestones and deliverables, preventing misunderstandings and streamlining development.

Wireframes

Wireframes were created early in the process to map out user flows, page hierarchy, and interface components. They provided a visual blueprint for both designers and developers, making it easier to identify potential usability issues before coding began. Wireframes were refined iteratively based on team feedback to ensure clarity and functionality.

Dev Plan

The development plan divided tasks into front-end, back-end, and integration work. Each module had clearly defined responsibilities, deadlines, and review points. Agile methodology guided our sprints, allowing the team to address bugs, optimize features, and implement improvements efficiently. This structured approach ensured the project progressed smoothly and met the intended goals.
